## [0.1.0] - 2026-02-26

### Added
- Initial release of `pi-web-search` as a Pi extension package.
- `web_search` tool powered by Perplexity Search API, with support for query, result limits, domain allow/block filters, and recency filtering.
- `web_fetch` tool for fetching a known URL and extracting readable text content.
- Package metadata for Pi extension discovery (`"keywords": ["pi-package"]` and `"pi.extensions"`).
- Project documentation in `README.md`.
